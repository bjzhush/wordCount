wordCount
=========

Count and analyze frequency of words in given contents

本工具主要用于辅助英语学习
比如我正在通过BBC或者CNN学习英语,必然有很多单词我不懂,传统的分析办法是遇到一个背诵一个,本程序的设计初衷,
通过分析已经阅读过的文章及爬虫爬取(比如指定BBC某些分类的文章),然后分析单词出现的频率,来有重点地对高频生词进行学习,从而达到最大化的效果
